## GetYourCrown
멀티플레이 캐주얼 왕관 뺏기 게임
<br>
<br>

## 프로젝트 개요
GetYourCrown은 폴가이즈의 꼬리잡기와 롤토체스의 증강 시스템에서 아이디어를 얻은 게임입니다.  
왕관을 유지해야 하는 단순한 목표 위에 게임 도중 등장하는 증강 요소를 더해  
매 판 다른 전략적 선택이 가능하도록 설계했습니다.
<br>
<br>

## 역할
- Photon Network 기반 대기실 멀티플레이 동기화 구현
- 채팅 및 캐릭터 선택 상태 실시간 동기화
- Unity Cloud Save를 활용한 유저 데이터 저장 및 로드
<br>
